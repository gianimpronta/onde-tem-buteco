# Política de Segurança

## Versões suportadas

Este projeto é um trabalho em andamento. Apenas a versão mais recente na branch `main` recebe correções de segurança.

## Reportando uma vulnerabilidade

Se você encontrar uma vulnerabilidade de segurança, **não abra uma issue pública**.

Envie um e-mail para **gian.impronta@gmail.com** com:

- Descrição do problema
- Passos para reproduzir
- Impacto potencial
- Sugestão de correção (opcional)

Você receberá uma resposta em até **7 dias úteis**. Se a vulnerabilidade for confirmada, trabalharemos em uma correção e daremos crédito no changelog, a menos que você prefira anonimato.

## Escopo

Este projeto é **fan-made**, sem fins comerciais. Os dados exibidos pertencem ao Comida di Buteco. Vulnerabilidades no site oficial devem ser reportadas diretamente a eles.
